# ARP Spoofing Project

## Description
This project demonstrates an ARP Spoofing attack in a controlled virtual environment using Kali Linux and Ubuntu.

## Tools Used
- Oracle VirtualBox
- Kali Linux
- Ubuntu Linux
- arpspoof (dsniff)
- Terminal

## Objective
To understand how ARP protocol vulnerabilities can be exploited to perform a Man-in-the-Middle attack.

## Note
This project was performed strictly for academic and educational purposes.
